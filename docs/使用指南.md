# 使用指南

## 🌐 网站功能

### 主页功能

**6个AI动态模块**：
- 🌍 全球AI动态
- 🇨🇳 中国AI动态
- 🤖 大模型动态
- 🎬 视频模型动态
- 💻 AI编程工具进化
- ⚡ 当前主流应用

**交互功能**：
- 每个模块默认显示最新10条新闻
- 点击"查看更多"展开该模块所有新闻
- 点击新闻卡片跳转到原文链接
- 自动显示中英文标识

### 管理后台

访问地址：`https://你的网站地址/admin.html`

**功能**：
- 🔐 密码登录
- 📊 查看所有模块数据
- 🗑️ 删除单条新闻
- 🧹 清空某个模块
- 🔗 跳转原文

**默认密码**：`admin123456`（可在部署时修改）

## 🤖 数据来源

### 采集源

- **英文来源**：国际科技媒体（TechCrunch、The Verge等）
- **中文来源**：AI专业媒体（量子位、机器之心等）
- **搜索引擎**：Bing Search API

### 数据处理流程

1. **搜索**：根据关键词搜索最新新闻
2. **AI处理**：智谱AI翻译、解读、去水
3. **过滤**：自动过滤违规内容和无关内容
4. **去重**：根据标题相似度去重
5. **保存**：存储到GitHub仓库

### 更新机制

- **首次**：加载最新100条/模块
- **后续**：每4小时更新60条/模块
- **历史数据**：逐步回填
- **数据保留**：每个模块最多1000条

## 🎨 界面说明

### 莫兰迪色系

- **主色**：淡蓝灰 #A8B5C9
- **次要色**：淡灰蓝 #D7DDE2
- **强调色**：淡紫 #E6E6FA
- **文字色**：深灰蓝 #5A6A7A

### 布局结构

- **Bento Box网格**：2行3列
- **响应式设计**：手机端自适应
- **卡片式布局**：每个模块一个卡片
- **序号标识**：每条新闻带编号

### 新闻卡片信息

- **序号**：1-10（默认显示范围）
- **标题**：中文标题（英文已翻译）
- **摘要**：AI解读的摘要（50字以内）
- **时间**：发布时间
- **来源**：源网站标识
- **语言标识**：中文/英文标签

## ⚙️ 个性化配置

### 修改主题色

编辑 `src/styles/global.css`：

```css
:root {
  --morandi-primary: #A8B5C9;    /* 修改为你的颜色 */
  --morandi-secondary: #D7DDE2;
  --morandi-accent: #E6E6FA;
  /* ... */
}
```

### 修改布局

编辑 `src/components/NewsGrid.vue` 中的网格配置：

```vue
<el-col
  :xs="24"  <!-- 手机端：1列 -->
  :sm="12"  <!-- 平板：2列 -->
  :md="8"   <!-- 桌面：3列 -->
  :lg="8"
  :xl="8"
>
```
